# Cheat Sheet

## Basic Linux Commands.

### mkdir

Creates new directories.

Usage:   `mkdir <dir>` <br>
Example: `mkdir ~/cmds_cs` 

---

### cd

Chages your current working directory.

Usage:   `cd <dir>` <br>
Example: `cd ~/cmds_cs`

---

### touch

Creates empty file.

Usage: `touch <dir/file>` <br>
Exame: `touch README.md`

---
